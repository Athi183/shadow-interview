# Shadow Interview

Shadow Interview is a Chrome extension foundation for turning LeetCode practice
into a structured technical-interview experience. Milestone 1 supplies a dark,
floating launcher and local problem-context panel on LeetCode problem pages.
It deliberately contains no AI logic, backend calls, or data collection.

## Repository layout

- `frontend/` — reserved for a future standalone web interface.
- `backend/` — reserved for a future API and service layer.
- `extension/` — a Manifest V3 Chrome extension that adds a local sidebar on
  LeetCode.
- `docs/` — architecture and product notes.
- `assets/` — future visual and brand assets.

See [extension/README.md](extension/README.md) to load the extension locally.
